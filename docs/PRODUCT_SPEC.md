# Product Spec

AI Image Layer Extractor Pro is a local desktop tool for turning flat AI
generated PNG/JPG/WEBP images into editable transparent layer assets.

Primary workflows:

- Manual box selection with OpenCV GrabCut.
- Natural-language commands that produce structured local edit plans.
- Batch export to game/UI production sizes.
- Optional target extraction pipeline with pluggable detectors and segmenters.
- UE UMG export package generation.
- PSD-compatible package export.
- Debug-friendly `.ailp` project package save/open.

The product must continue to work offline without API keys or large model
weights.
